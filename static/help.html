Keyboard Shortcuts

Open in new window Close

Command Line

RETURN
run command line

Completion

TAB
start completion
↓ or TAB
next completion
↑ or ALT + TAB
prior completion
RETURN or typing
accept completion
=
enter all completions
ESC
cancel completion

History

CTRL + R or ALT + /
start history search from most recent,
then continue backward
CTRL + S or ALT + \
start history search from oldest,
then continue forward
↑ or ALT + ↑
prior match
↓ or ALT + ↓
next match
ESC or CTRL + G
cancel search
RETURN
reenter current match into command line
CTRL + U
clear search field

Navigation

↑ or ALT + ↑
prior job
↓ or ALT + ↓
later job
ALT + ←
focus on command line
ALT + →
focus on current job
RETURN
when focused on prior job, reenter job into command line

Current Job

PAGE DOWN | PAGE UP
scroll output down/up a page
SPACE | SHIFT + SPACE
scroll output down/up a page
HOME
scroll output to start
END
scroll output to end
— ALT can be added to any of the above.
ALT + 0
hide output
ALT + 1
resize output small (3 lines)
ALT + 2
resize output page
ALT + 3
resize output full
CTRL + D
send EOF
CTRL + C
interrupt (signal 2)
CTRL + \
quit (signal 3)
CTRL + 9
kill (signal 9)
